# Corpse Extended

A NeoForge compatibility mod for Minecraft 1.21.1 that extends the [Corpse](https://modrinth.com/mod/corpse) mod.

> **Requires the Corpse mod** - this is not a fork, and it cannot run without the original mod installed.


## Requirements

| Dependency | Version |
|---|---|
| Minecraft | 1.21.1 |
| NeoForge | 21.1.x |
| [Corpse](https://modrinth.com/mod/corpse) | 1.21.1-1.1.13+ |
